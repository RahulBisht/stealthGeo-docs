## Frequently Asked Questions (FAQ)

### Getting Started

**Q: How do I install StealthGeo?**  
A: Install StealthGeo from the Chrome Web Store. Once installed, click the extension icon in your browser toolbar to get started.

**Q: Is StealthGeo free?**  
A: Yes! StealthGeo is completely free and open source.

**Q: What browsers are supported?**  
A: StealthGeo currently supports Chrome and Chromium-based browsers (Edge, Brave, Opera, etc.).

---

### Using the Extension

**Q: How do I spoof my location?**  
A: 
1. Click the StealthGeo extension icon
2. Enter coordinates or search for an address
3. Click "Set Location" to activate spoofing
4. Refresh the webpage to apply the new location

**Q: Can I save favorite locations?**  
A: Yes! After setting a location, click the "Save" button to add it to your favorites for quick access later.

**Q: How do I turn off location spoofing?**  
A: Click the StealthGeo icon and select "Reset to Real Location" or disable the extension.

**Q: Why isn't the location changing on some websites?**  
A: Some websites may cache location data. Try:
- Clearing your browser cache
- Opening the site in an incognito/private window
- Restarting your browser

### Troubleshooting

**Problem: The extension isn't working**  
**Solution:**
- Ensure the extension is enabled in `chrome://extensions/`
- Check that you've granted necessary permissions
- Try disabling and re-enabling the extension
- Restart your browser

**Problem: Location not updating on websites**  
**Solution:**
- Refresh the webpage after setting a new location
- Clear browser cache and cookies
- Some websites may detect spoofing - this is expected behavior

**Problem: Extension icon not showing**  
**Solution:**
- Pin the extension: Click the puzzle icon in Chrome toolbar ‚Üí Find StealthGeo ‚Üí Click the pin icon
- Check if the extension is enabled in `chrome://extensions/`

**Problem: Coordinates not working**  
**Solution:**
- Ensure coordinates are in the correct format: `latitude, longitude`
- Latitude range: -90 to 90
- Longitude range: -180 to 180
- Example: `40.7128, -74.0060` (New York City)

## Privacy & Security

**Q: Does StealthGeo collect my data?**  
A: No. StealthGeo does not collect, store, or transmit any personal data. All settings are stored locally on your device. See our [Privacy Policy](/privacy-policy) for details.

**Q: Is it safe to use?**  
A: Yes. StealthGeo is open source, and you can review the code on GitHub. We don't access your real location or any personal information.

## Technical Support

### System Requirements
- Chrome version 88 or higher
- Chromium-based browsers (Edge, Brave, Opera)
- Active internet connection for address search

### Permissions Explained
- **debugger**: Required for advanced location spoofing
- **geolocation**: To override browser location API
- **storage**: To save your preferences locally
- **privacy**: To modify privacy settings for effective spoofing
- **<all_urls>**: To inject spoofing scripts on websites
